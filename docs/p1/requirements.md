## 1. 역할 정의

| 역할 | 설명 |
|------|------|
| GUEST | 비회원 — 강의 목록·상세 조회 가능 |
| USER | 수강생 — 구매·수강·리뷰 등 전체 기능 |
| INSTRUCTOR | 강사 — 강의 개설·관리 (P2) |
| TUTOR | 원어민 튜터 — 1:1 수업 제공 (P2) |
| ADMIN | 관리자 — 전체 플랫폼 관리 (P3) |

## 3. 비기능 요구사항 (NFR)

| ID | 요구사항 | 수용 수준 |
|----|----------|-----------|
| NFR-001 | API 응답 시간 | 일반 API 200ms 이내 |
| NFR-002 | 영상 스트리밍 지연 | 재생 시작 2초 이내 |
| NFR-003 | 가용성 | 99.5% 이상 |
| NFR-004 | 보안 | HTTPS 필수, 비밀번호 bcrypt 암호화, JWT 15분 만료 |
| NFR-005 | 반응형 | 데스크탑·태블릿·모바일 웹 지원 |

## 4. 제약사항

- 서비스 형태: 웹 전용 (모바일 앱 제외)
- 영상 저장: AWS S3 + CloudFront CDN
- P1 단계는 수강생 기능만 구현, 강사·튜터·관리자는 P2~P3에서 구현
