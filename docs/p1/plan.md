## 1. 배경 및 목적

글로벌화로 외국어 능력 수요가 증가하지만, 기존 학습 환경에는 한계가 있다.

- 원어민 과외: 높은 비용으로 접근성 낮음
- AI 학습 앱: 체계적 커리큘럼 없이 단편적
- 분절된 경험: 강의·회화·튜터링이 각기 다른 플랫폼에 분산

LinguaClass는 **강의 수강 → AI 회화 연습 → 원어민 튜터 1:1 수업**으로 이어지는 학습 사이클을 하나의 플랫폼에서 제공한다.

## 3. 주요 기능

### 수강생 (P1 구현 범위)

| 기능 | 내용 |
|------|------|
| 강의 탐색 | 언어별·레벨별·카테고리별 필터, 키워드 검색, 정렬 |
| 강의 구매 | 단건 구매, 카드/카카오페이 결제, 환불 |

| 위시리스트 | 관심 강의 저장·관리 |
| 리뷰 | 강의 리뷰 작성 |

| 기능 | 단계 |
|------|------|
| AI 회화 연습 (발음·문법 피드백) | P2 |
| 원어민 튜터 1:1 예약·화상 수업 | P2 |
| 단어장, 학습 노트, Q&A | P2 |
